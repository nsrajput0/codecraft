HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements represented by tags. HTML elements are the building blocks of HTML pages. They are represented by tags, which are enclosed in angle brackets.

HTML Code Example

                    
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
    </html>

HTML Common Tags

Anchor Tag

Anchor tag is used to create hyperlinks in a web page. It can link to another page or a specific section within the same page.

                        
     <a href="yourlink">Click to toggle link</a>

Image Tag

Image tag is used to embed an image in a web page.

                        
    <img src="yourimage.jpg" alt="Description">

Paragraph Tag

paragraph tag is used to define a paragraph.

                        
    <p>This is a paragraph.</p>

Bold Tag

Bold tag is used to make text bold.

                        
    <b>This text is bold.</b>

Div Tag

Div tag is used to create a division or section in a web page. It is often used for layout purposes.

                        
    <div>
        <h2>This is a div</h2>
        <p>This is a paragraph inside a div.</p>
    </div>

Span Tag

Span tag is used to group inline elements in a document. It is often used to apply styles to a specific part of the text.

                        
    <span style="color: red;">This text is red.</span>

Pre Tag

Pre tag is used to display preformatted text. It preserves both spaces and line breaks.

                        
    <pre>
        <code>
            <h1>Hello, World!</h1>
        </code>
    </pre>

Comment Tag

Comment tag is used to insert comments in the HTML code. Comments are not displayed in the browser.

                        
    <!-- This is a comment -->

Underline Tag

Underline tag is used to underline text.

                        
    <u>This text is underlined.</u>

Italic Tag

Italic tag is used to italicize text.

                        
    <i>This text is italicized.</i>

Break Tag

Break tag is used to insert a line break.

                        
    <br>

Horizontal Rule Tag

Horizontal rule tag is used to create a thematic break in the content.

                        
    <hr>

Navigation Tag

Navigation tag is used to define a navigation section in a web page.

                        
    <nav>
        <ul>
            <li><a href="#home">Home</a></li>
            <li><a href="#about">About</a></li>
            <li><a href="#contact">Contact</a></li>
        </ul>
    </nav>

Section Tag

Section tag is used to define a section in a web page. It is often used to group related content.

                        
    <section>
        <h2>Section Title</h2>
        <p>This is a section of content.</p>
    </section>

Article Tag

Article tag is used to define an independent piece of content in a web page. It is often used for blog posts or news articles.

                        
    <article>
        <h2>Article Title</h2>
        <p>This is an article about a specific topic.</p>
    </article>

Footer Tag

Footer tag is used to define a footer for a web page or a section. It typically contains information about the author, copyright, or links to related documents.

                        
    <footer>
        <p>Copyright © 2023</p>
        <p>Links: <a href="#privacy">Privacy Policy</a> | <a href="#terms">Terms of Service</a></p>
    </footer>

Main Tag

Main tag is used to define the main content of a web page. It is typically used to wrap the primary content of the page.

                        
    <main>
        <h1>Main Content Title</h1>
        <p>This is the main content of the page.</p>
    </main>

Link Tag

Link tag is used to link external resources like stylesheets or icons.

                        
    <link rel="stylesheet" href="styles.css">
    <link rel="icon" type="image/x-icon" href="favicon.ico">

Address Tag

Address tag is used to define contact information for the author or owner of a document or an article.

                        
    <address>
        <p>Contact us at: <a href="mailto:info@example.com">info@example.com</a></p>
    </address>

Progress Tag

Progress tag is used to display the progress of a task.

                        
    <progress value="70" max="100">70%</progress>

Meter Tag

Meter tag is used to represent a scalar measurement within a known range.

                        
    <meter value="0.7" min="0" max="1">70%</meter>

Details Tag

Details tag is used to create a disclosure widget from which the user can obtain additional information or controls.

                        
    <details>
        <summary>More Info</summary>
        <p>Here is some additional information.</p>
    </details>

Code Tag

Code tag is used to define a piece of computer code.

                        
    <code>
        <p>This is a paragraph of code.</p>
    </code>

Input Tag

Input tag is used to create interactive controls in a web form that users can use to enter data.

                        
    <input type="text" placeholder="Enter text here">
    <input type="checkbox"> Check me</input>
    <input type="radio" name="options" value="1"> Option 1</input>
    <input type="radio" name="options" value="2"> Option 2</input>
    <input type="submit" value="Submit">

Body Tag

Body tag is used to define the main content of an HTML document.

                        
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is a paragraph of text in the body.</p>
    </body>

Heading Tag

Heading tags are used to define headings in a web page. There are six levels of headings, from <h1> to <h6>.

                        
    <h1>This is a heading</h1>
    <h2>This is a sub-heading</h2>
    <h3>This is a smaller heading</h3>

ul Tag

ul tag is unordered list tag used for making list in unordered format using symbols such as circle, square, disc.

                        
    <ul>
        <li>Item 1</li> 
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>

ol Tag

ol tag is ordered list tag used for making list in ordered format using numbers.

                        
    <ol>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ol>

li Tag

li tag is used to define a list item in an ordered or unordered list.

                        
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>

    <ol>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ol>
