CSS

CSS

CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. CSS controls the layout of multiple web pages all at once.

CSS is designed to enable the separation of presentation and content, elements are the building blocks of HTML pages. They are represented by tags, which are enclosed in angle brackets.

Tailwind CSS Properties

padding

The padding CSS property is a shorthand property for defining the space between the content of an element and its border. It can take one to four values, representing the padding for the top, right, bottom, and left sides of the element.

                        
     p-0  #padding from all sides
     p-4  #padding of 1rem (16px) on all sides
     p-x-4  #padding of 1rem (16px) on left and right sides
     p-y-4  #padding of 1rem (16px) on top and bottom sides

margin

The margin CSS property is a shorthand property for defining the space outside an element's border. It can take one to four values, representing the margin for the top, right, bottom, and left sides of the element.

                        
     m-0  #margin from all sides
     m-4  #margin of 1rem (16px) on all sides
     m-x-4  #margin of 1rem (16px) on left and right sides
     m-y-4  #margin of 1rem (16px) on top and bottom sides

background-color

The background-color CSS property sets the background color of an element. It can take any valid CSS color value, including named colors, hex codes, RGB, RGBA, HSL, and HSLA values.

                        
     bg-transparent  #make background transparent
     bg-black  #set background to black
     bg-white  #set background to white
     bg-red-500  #set background to a shade of red

text-color

The text-color CSS property sets the color of the text content of an element. It can take any valid CSS color value, including named colors, hex codes, RGB, RGBA, HSL, and HSLA values.

                        
     text-transparent  #make text transparent
     text-black  #set text to black
     text-white  #set text to white
     text-red-500  #set text to a shade of red

border

The border CSS property is a shorthand property for setting the width, style, and color of an element's border. It can take one to three values, representing the width, style, and color of the border.

                        
     border-0  #remove border
     border-2  #set border width to 2px
     border-dashed  #set border style to dashed
     border-red-500  #set border color to a shade of red

flex

The flex CSS property is a shorthand property for setting the flex-grow, flex-shrink, and flex-basis properties of a flex item. It allows you to control how a flex item grows or shrinks within a flex container.

                        
     flex-1  #allow flex item to grow and fill available space
     flex-auto  #allow flex item to grow and shrink as needed
     flex-initial  #set flex item to its initial size
     flex-none  #prevent flex item from growing or shrinking

grid

The grid CSS property is a shorthand property for setting the grid-template-columns, grid-template-rows, and grid-template-areas properties of a grid container. It allows you to define the structure of a grid layout.

                        
     grid-cols-1  #create a single column grid
     grid-cols-2  #create a two-column grid
     grid-cols-3  #create a three-column grid
     grid-rows-2  #create a two-row grid

shadow

The shadow CSS property is a shorthand property for setting the box-shadow and text-shadow properties of an element. It allows you to add shadows to elements, enhancing their visual appearance.

                        
     shadow-sm  #apply a small shadow
     shadow-md  #apply a medium shadow
     shadow-lg  #apply a large shadow
     shadow-xl  #apply an extra-large shadow

rounded

The rounded CSS property is a shorthand property for setting the border-radius of an element. It allows you to create rounded corners on elements, enhancing their visual appearance.

                        
     rounded-sm  #apply a small border radius
     rounded-md  #apply a medium border radius
     rounded-lg  #apply a large border radius
     rounded-xl  #apply an extra-large border radius

transition

The transition CSS property is a shorthand property for setting the transition-property, transition-duration, transition-timing-function, and transition-delay properties of an element. It allows you to create smooth transitions between different states of an element.

                        
     transition-all  #apply transition to all properties
     transition-colors  #apply transition to color properties
     transition-opacity  #apply transition to opacity
     transition-transform  #apply transition to transform

opacity

The opacity CSS property sets the opacity level for an element, making it transparent or semi-transparent. It can take a value between 0 (completely transparent) and 1 (completely opaque).

                        
     opacity-0  #completely transparent
     opacity-25  #25% transparent
     opacity-50  #50% transparent
     opacity-75  #75% transparent
     opacity-100  #completely opaque

display

The display CSS property sets the display behavior of an element. It can take values like block, inline, flex, grid, and more, determining how the element is rendered in the layout.

                        
     block  #display as a block element
     inline  #display as an inline element
     flex  #display as a flex container
     grid  #display as a grid container

overflow

The overflow CSS property controls what happens to content that is too large to fit in an area. It can take values like visible, hidden, scroll, and auto.

                        
     overflow-visible  #content is not clipped and may be rendered outside the element's box
     overflow-hidden  #content is clipped and not visible
     overflow-scroll  #content is clipped but a scrollbar is added
     overflow-auto  #browser decides whether to add a scrollbar

position

The position CSS property sets how an element is positioned in the document. It can take values like static, relative, absolute, fixed, and sticky.

                        
     position-static  #default positioning
     position-relative  #position relative to its normal position
     position-absolute  #position relative to the nearest positioned ancestor
     position-fixed  #position relative to the viewport
     position-sticky  #toggle between relative and fixed, depending on scroll position

z-index

The z-index CSS property sets the stack order of an element. Elements with a higher z-index are displayed in front of those with a lower z-index.

                        
        z-0  #default z-index
        z-10  #z-index 10
        z-20  #z-index 20
        z-30  #z-index 30
        z-40  #z-index 40
        z-50  #z-index 50

CSS Common Properties

Padding

Padding is the space between the content of an element and its border. 

                        
     padding: 16px;
     padding-top: 16px;
     padding-right: 16px;
     padding-bottom: 16px;
     padding-left: 16px;

Margin

Margin is the space outside the border of an element.

                        
     margin: 16px;
     margin-top: 16px;
     margin-right: 16px;
     margin-bottom: 16px;
     margin-left: 16px;

Background Color

Background color sets the background color of an element.

                        
     background-color: #f0f0f0;

Text Color

Text color sets the color of the text content of an element.

                        
     color: #333333;

Border

Border sets the width, style, and color of an element's border.

                        
     border: 1px solid #333333;
     border-top: 1px solid #333333;
     border-right: 1px solid #333333;
     border-bottom: 1px solid #333333;
     border-left: 1px solid #333333;

Flex

Flex properties control the layout of flex items within a flex container.

                        
     display: flex;
     flex-direction: row;
     flex-wrap: nowrap;
     justify-content: flex-start;
     align-items: stretch;

Grid

Grid properties control the layout of grid items within a grid container.

                        
     display: grid;
     grid-template-columns: repeat(3, 1fr);
     grid-gap: 16px;

Shadow

Shadow adds a shadow effect to an element.

                        
     box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
        text-shadow: 1px 1px 2px rgba(0, 0, 0, 0.2);

Rounded

Rounded corners for elements.

                            
     border-radius: 4px;

Transition

Transition properties for smooth animations.

                        
     transition: all 0.3s ease;

Transform

Transform properties for 2D/3D transformations.

                        
     transform: translateX(100px) rotate(45deg);

Opacity

Opacity sets the transparency level of an element.

                        
     opacity: 0.5;

Display

Display properties control how an element is displayed in the layout.

                        
     display: block;
    display: inline;
    display: flex;
    display: grid;
    display: contents;
    display: none;

Overflow

Overflow properties control how content that overflows an element is handled.

                        
     overflow: visible;
    overflow: hidden;
    overflow: scroll;
    overflow: auto;
    overflow: clip;

Position

Position properties control how an element is positioned in the layout.

                        
     position: static;
    position: relative;
    position: absolute;
    position: fixed;
    position: sticky;

Z-Index

Z-index controls the stacking order of positioned elements.

                        
     z-index: 1;
    z-index: 2;
    z-index: 3;
    z-index: 4;
    z-index: 5;

Custom Properties

Custom properties (CSS variables) allow you to store values for reuse.

                        
     --main-bg-color: #3490dc;
    --main-text-color: #333;
    --font-stack: 'Helvetica Neue', sans-serif;
